YouTube Scrape V1.0 Beta

https://rr-scrape.herokuapp.com/

Utilizes two main tools to complete the task. There is an indipendant NodeJS script that runs in terminal against a CSV list of provided YouTube viedo IDs

After scraping the data from the supplied CSV, the front-end tool is deployed to view the data. These will eventually be integrated together, and user will be able to upload new files for processing directly from within the app.

There are several dependencies within NodeJS that must be installed in order to utilize the tool locally on your computer. To navigate the issue, the app has been deployed to a free Heroku instance, where it can be accessed by anyone using this link

Dev Stack

  • Ionic Framework - UI

    Local or Heroku

  • Nodejs - Data Scraper

    Local only. In development to integrate with front-end, making the scraper accessible from Heroku as well.

Source Control

The course files for this project are held within a private GitHub repo and can be shared or forked by a contributer who is granted access. The files can also be moved at any time to another repo that is controlled by the client.

https://github.com/AlphaNerd2021?tab=repositories

To update and push code to GitHub:

  • $ git add .
  • git commit -m "[some commnets here]"
  • git push

Deployment

The app is automatically deployed to Heroku once it is updated in GitHub. Very easy and no hassle. Heroku also handles all needed SSL security certs to maintain a secure interface (overkill at the moment, but will allow for features to be added later that allow users to upload files etc.

  • Run Locally

    $ git clone [repo-address-here]

    $ cd YouTube-Scraper

    * if needed run $ nvm use 14.15

    $ npm install This will innstall all required dependencies to run the application in your browser, from your computer (aka: locally). You only need to do this the first time you download the app, before you run it.

    $ npm start This will run the front-end application in your browser.
